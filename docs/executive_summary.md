# Executive Summary

## Project title

**Supply Chain & Warehouse Optimization: Pallet Loading Under Storage Time and Relative Humidity**

## Summary

This project demonstrates how pallet loading can be optimized under real warehouse conditions. The research considers box dimensions, pallet dimensions, interlock stacking, relative humidity, storage time, maximum allowable pallet height, maximum allowable pallet weight, and dynamic compression strength.

## Business problem

Warehouses need to maximize pallet utilization while preventing box damage and pallet instability. Traditional pallet loading approaches often focus on space utilization, but environmental and storage conditions can weaken boxes and change safe stacking decisions.

## Proposed solution

The paper proposes a two-phase heuristic algorithm:

1. Generate horizontal layer configurations using block techniques.
2. Optimize the maximum number of boxes per pallet using a mathematical model with real-world constraints.

## Key result

The approach was tested on 52 real-life DHL Supply Chain datasets and demonstrated how mixed horizontal layer configurations can improve pallet utilization while considering pallet stability.

## Consulting relevance

The method supports better warehouse layout decisions, safer palletization policies, reduced damage risk, and improved logistics performance.
