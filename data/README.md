# Data Notes

This folder contains summary and demonstration datasets prepared from the published pallet-loading optimization paper.

## Included files

### `storage_time_factors.csv`
Storage-time coefficients used to adjust dynamic compression strength.

### `relative_humidity_factors.csv`
Relative-humidity coefficients used to adjust dynamic compression strength.

### `other_dynamic_strength_factors.csv`
Orientation, interlock, and pallet-surface factors used in the dynamic compression strength calculation.

### `selected_pvu_examples.csv`
Selected pallet volume utilization and number-of-boxes examples based on reported values in the paper.

### `selected_humidity_impact_examples.csv`
Selected examples showing how load height changes under different relative humidity levels.

### `selected_storage_time_impact_examples.csv`
Selected examples showing how load height changes under different storage times.

### `case_study_summary.csv`
A concise summary of the project context and methodology.

## Note

The included files are summary/demo datasets prepared for portfolio and educational purposes based on reported values from the paper. They are not a replacement for the full published paper.
